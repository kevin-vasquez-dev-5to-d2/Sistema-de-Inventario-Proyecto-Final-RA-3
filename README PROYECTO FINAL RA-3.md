# Sistema de Inventario

## Descripción:

Este proyecto consiste en una aplicación de escritorio desarrollada en C# que permite gestionar el inventario de un negocio de forma sencilla y organizada. El sistema permite registrar productos, categorías, proveedores, usuarios y clientes, además de controlar las entradas y salidas de mercancía. También incluye la generación de reportes para visualizar los movimientos realizados y un formulario aparte para visualizar los movimientos anteriormente creados y crear reportes en base a estos.

---

## Tecnologías Utilizadas:

* **Lenguaje:** C#
* **Herramientas / Framework:** Windows Forms (.NET Framework)
* **Base de datos:** SQL Server 2022.
* **Entorno o IDE:** Visual Studio 2026.

## Uso o Ejecución:

1. Abrir el proyecto en Visual Studio.
2. Ejecutar el sistema presionando F5.
3. Iniciar sesión con un usuario registrado.
4. Registrar categorías, productos, proveedores y clientes.
5. Realizar movimientos de entrada y salida.
6. Generar reportes desde el sistema.
